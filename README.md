# Congress 101

A hip, no-BS crash course in the US Congress — built for high school government students.

**Live site:** coming once GitHub Pages is enabled ↓

## What's inside

- **Basics** — House vs Senate side-by-side, leadership cheat sheet
- **Meet Your Reps** — CA Senators + ZIP-to-House-rep lookup
- **How a Bill Becomes a Law** — 7-step pipeline with real-talk commentary
- **Bill Ordering Mini-Game** — drag-free click-in-order puzzle
- **Powers of Congress** — expressed, non-legislative, elastic clause
- **Gerrymandering Deep Dive** — California district map, origin story (1812!), packing vs cracking visualized
- **Pork Hall of Shame** — 9 real, ridiculous earmarks (yes, the Teapot Museum is real)
- **Lobbying Deep Dive** — top-spending industries, the revolving door, the Abramoff case study
- **The Money** — live-ticking $39T national debt counter + federal budget breakdown
- **Happening Now** — real 119th Congress bills pending in 2026
- **Vocab** — 20 flip-cards
- **Quiz** — 10 questions with scoring

## Running it locally

Just open `index.html` in any modern browser. No install, no build, no server required.

```
cd congress-site
start index.html   # Windows
open index.html    # Mac
```

## Hosting on GitHub Pages

Go to **Settings → Pages → Build from `main` branch, root folder**. That's it.

## A note about videos

The site references two local video files that live in the **parent folder** (not in this repo — they're too big for GitHub):

- `Gerrymandering Explained.mp4`
- `Jack Abramoff- The lobbyist's playbook.mp4`

Videos will play when you open the site locally as long as those files are one folder up from `index.html`. On the hosted GitHub Pages version, the video embeds will fall back to a graceful "watch in class" message.
